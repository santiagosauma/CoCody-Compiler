**Descripcion:** Este ejercicio te pide que implementes una funcion que realice una operacion especifica sobre una matriz. 

**Problema:**

Escribe una funcion en Python llamada `rotacion_matriz(matriz, direccion)` que reciba una matriz bidimensional y una cadena que representa la direccion de rotacion ( "izquierda" o "derecha" ) como parametros. La funcion debe rotar la matriz en la direccion especificada y devolver la matriz rotada.

**Ejemplo:**

**Entrada:**

```python
matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
direccion = "izquierda"
```

**Salida:**

```python
[
    [3, 6, 9],
    [2, 5, 8],
    [1, 4, 7]
]
```

**Recuerda:**

* La funcion debe modificar la matriz original en lugar de crear una nueva.
* El codigo debe ser eficiente y manejar matrices de cualquier tamano.
* La direccion de rotacion puede ser "izquierda" o "derecha".
